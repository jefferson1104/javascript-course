# Operators

##### Arithmetic operators
addition ( + )
subtraction( - )
multiplication( * )
division ( / )

Examples:
```js
console.log(2 + 2);
console.log(2 - 2);
console.log(2 * 3);
console.log(2 / 2);
```

##### Comparison operators
greater than ( > )  
less than ( < )  
equal to ( == ) 
no equal ( != ) 

Examples:
```js
console.log(5 > 2);
console.log(2 < 4);
console.log(10 == 10);
console.log(30 != 36);
```

##### Logical Operators AND (&&)
check for both conditions to be true
```js
a > 5 && a < 10
```

##### Logical Operators OR (||)
check for at least one conditions to be true
```js
a > 5 || a > 10
```

##### Logical Operators NOT (!)
returns false if the result is true
```js
!(a>5)
```
